Main File Structure

All the files are well organized, it's so easy to work with the template. Unzip the 'main-files' file. Open main-file folder. You will find the 15 html files. Also you will see 4 directories (css, fonts, images and js) inside 'assets' folder. In the 'css' folder, you will find 'style.css' and other styling files. The 'fonts' folder contains essential font files for the template. The 'images' folder contains all images for the template. In the 'js' folder, you will find 'main.js' and other jQuery library files.
